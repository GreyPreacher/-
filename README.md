# 贪吃蛇
学校的c语言课程的贪吃蛇大作业，用VS2010和Easyx写的，十分简单的一个小项目，代码行数也不多，项目中无绝对目录设置，只有相对目录，直接使用即可  
该项目由张宵霆和肖滃然一起完成，许多年后会有看会感叹真是一段美好的回忆
